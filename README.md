# 背景
2050年カーボンニュートラルの達成には、化石資源依存から社会内の蓄積炭素を資源とする炭素循環への転換が不可欠である。リサイクルの高度化が進む一方で、世界の資源投入量とCO₂排出量は増加し続けている。脱炭素社会への移行を加速させるためには、社会動態と技術ポテンシャルを同時に考慮し、システム変革の転換点を導く独自の設計指針が必要である。

# 課題
- ①需要動態と滞留時間による資源削減効果の減殺<br>
資源需要の拡大や製品寿命の分布が、循環による資源削減効果を物理的に制限するメカニズムが解明されていない。
- ②全体評価と影響因子特定の分断<br>
マクロな社会推計とミクロな技術特性を接続する評価基盤がなく、システム全体のパフォーマンスを左右する支配的因子を特定できない。
- ③社会実装に向けたシステム転換点の未特定<br>
技術的な理想ルートは示せても、それを実現するために必要なコスト、技術水準、政策等の具体的な境界条件が定量的に示されていない。
# 目的
- 廃プラスチックを対象に、炭素循環効率の最大化とバージン材投入量の最小化を両立する理想的な移行経路を解明し、その実現に不可欠な技術、政策的境界条件を提示することを目的とする。
# 研究項目
1. **資源需要と循環供給の関係を記述する評価基盤の構築**<br>
将来の社会動態(需要、寿命など)と技術選択が相互に影響し合うフィードバック構造をモデル化し、循環システムの実効性を定量化する。
   - 時間的・質的制約の定式化<br>
   製品寿命による資源回収の時間的遅れと、リサイクルに伴う質の劣化が、将来の資源供給能力を制限する物理的制約をモデルに組み込む。
   - 再帰的な最適化によるフィードバックの実装<br>
   年次ごとの廃プラスチック発生予測を制約条件とし、翌年の資源投入量を最小化するポートフォリオを逐次決定する動的な計算アルゴリズムを開発する。
2. **感度解析による資源循環の支配的因子の特定**<br>
構築したモデルを用い、複雑なサプライチェーンの中でどの変数が資源削減にもっとも寄与するか、その因果のメカニズムを解明する。
- 需要因子と技術因子の総合的な評価<br>
製品寿命の長短と、選別・リサイクルの精緻さが、最終的な資源消費削減に与える影響を同一指標上で比較定量化する。
- ボトルネック、プロセスの抽出<br>
  資源循環を阻害している特定のプロセスや製品カテゴリーを特定し、重点的に改善すべき対象を化学的に根拠づける。
3. **理想的な炭素循環へ移行するための転換点の提示**<br>
現状の処理構造から理想的な構造へ社会が転換するために必要な、具体的かつ実現可能な境界条件を導出する。<br>
- 多角的な将来シナリオ下での最適移行ルートの提示<br>
需要増や新規技術導入などの不確実な将来変化を踏まえて、目指すべきリサイクル構成の時系列的な変遷を明らかにする。
- 社会実装を可能にする境界条件の特定<br>
理想ルートが成立するために必要な技術革新目標、コスト低減幅などの閾値を定量的に算出し、具体的な政策、経営指針として提示する。
