# 背景
低炭素・循環型経済の実現は、気候変動や資源枯渇、廃棄物問題の解決に向けて不可欠である。しかし、再生可能エネルギー導入やリサイクルの高度化が進む一方で、2024年のエネルギー期限CO₂排出量は過去最高となり、二次材料の利用率も低下している。すなわち、低炭素化及び資源循環の方向性は示されているものの、その進展の速度と規模は不十分である。特に、資源循環は再資源化効率の向上に依存しており、資源需要の増加に十分対応できていない。

# 課題
- ①資源需要と循環効率を統合した評価・設計の欠如<br>
資源消費削減に向けてリサイクルの高度化が進められてきたが、これは資源利用効率の改善にとどまり、資源湯用の増加との関係は十分に考慮されていない。また、資源の流れや蓄積は既存の分析手法により把握可能であるが、それらは主にトップダウン的な評価にとどまり、どの循環ルートや技術選択が資源需要および循環効率に影響を与えるかを特定することが困難である。その結果、循環効率が向上しても総需要が増加すれば資源消費量は増加し得るほか、循環の滞留時間によっては資源投入削減効果が遅れて現れる。
- ②将来変化および実現可能性を考慮した資源循環の検討不足<br>
資源需要や循環効率は時間とともに変化するため、特定時点での評価のみでは望ましい資源循環の在り方を示すことはできない。また、りそうてきな　循環ルートが明らかになったとしても、その実現に必要な条件(コストや制約)は十分に検討されていない。

# 目的
- 資源需要と資源循環の関係を統合的に捉え、資源消費量と循環効率を同時に評価した上で、将来条件を考慮した資源循環ルートの最適化手法を構築することを目的とする。さらに、廃プラスチックを対象として、資源消費量の最小化と資源効率の向上を両立する循環ルートを明らかにするとともに、その実現に必要な条件を提示する。

# 研究項目
1. **資源需要と循環効率を統合した資源フローおよび影響院試の解析**
   - 資源消費量(累積バージン材投入量)および循環効率(再資源化、滞留時間など)の定義<br>
   - 資源需要および製品ストックのモデル化<br>
   - 資源フローの時間変化の表現<br>
   - 選別・リサイクル技術が資源消費量及び循環効率に与える影響の解析
2. **将来条件下における資源循環ルートの最適化と実現条件の導出**
   - 資源需要および循環条件のシナリオ設定(需要増+50%,MRメイン、CRメイン、全ERなど)<br>
   - バージン材消費量と循環効率を目的関数とした最適化<br>
   - 目的関数に影響を与える因子の特定<br>
   - 実現に必要な条件(技術・コスト・政策)の分析
